# Week 1 – SIEM Implementation and Security Monitoring using Wazuh

## Overview
This project demonstrates implementation of a SIEM platform using Wazuh in a virtual SOC lab. A Windows endpoint was monitored by Wazuh, failed login events were generated, alerts were detected, and incidents were analyzed.

## Objectives
- Deploy Wazuh SIEM
- Connect Windows endpoint
- Simulate brute-force login attempts
- Detect Event ID 4625
- Map alerts to MITRE ATT&CK
- Analyze SOC workflow

## Lab Setup
- Ubuntu VM: Wazuh Server
- Windows 11 VM: Endpoint
- VMware Workstation

## Key Events
- Event ID 4625 Failed Login
- Rule ID 60122 Login Failure
- Rule ID 60204 Multiple Login Failures

## Skills Demonstrated
- SIEM Monitoring
- Threat Detection
- Log Analysis
- Alert Investigation
- SOC Operations
